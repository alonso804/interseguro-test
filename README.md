# Interseguro - Prueba Técnica

## Usar versión de node

```bash
nvm use
```

## Instalar dependencias

```bash
pnpm install --frozen-lockfile
```

## Correr en desarrollo

```bash
pnpm dev
```

## Build al repositorio

```bash
pnpm build
```

## Correr en producción

```bash
pnpm start
```

## Correr test

```bash
pnpm test
```
